<h1 align="center">Database / Migrations 🗄️</h1>

Repository containing database schemas and migrations used by the services. It
includes:

- Drizzle schema definitions in `database/src/`.
- SQL migrations in `database/migration/` (generated by Drizzle).
- Utilities to apply migrations during service startup (Bun).

> Generally, the authentication service applies migrations on startup; the
> remaining services should rely on/validate against the centralized migration
> history.
